## ФУНКЦИОНАЛ: продуктовые идеи (что умеют их сайты)

1. **Онлайн-конструктор макетов с шаблонами** (kopirka, copy.ru — у обоих в H1). Библиотека шаблонов **по 26 отраслям** (kopirka) = заодно SEO-сетка «визитки для медицины/недвижимости/юристов…». Личный кабинет с сохранёнными макетами.
2. **Загрузка макета с автопроверкой** «на техническую корректность» (copy.ru) — снижает страх заказа.
3. **Три пути к заказу на одной странице** (copy.ru): загрузить макет / онлайн-редактор / заказать дизайн (платно). У нас только калькулятор.
4. **Скидка 5% за онлайн-заказ** (copy.ru) — прямо в калькуляторе, стимул не звонить.
5. **Срочность как опция калькулятора**: стандарт 24 ч / срочно до 4 ч (copy.ru) — у нас «срочно» только как кластер-посадочная.
6. **Брендированные материалы в калькуляторе**: «Плёнка Oracal 85 г/м², Raflatac, Muflon» (copy.ru) — конкретика брендов повышает доверие. У нас материалы обезличенные.
7. **Блок доставки с ценами прямо на странице продукта** (copy.ru): самовывоз/ПВЗ/курьер с рублями и сроками.
8. **Тестовые визитки** (kopirka: `/catalog/testovye-vizitki/`) — пробный мини-тираж как продукт-крючок.
9. **Телеграм-бот для заказа** (copy.ru).
10. **Блог-статьи, привязанные к продукту** («Наклейки с плоттерной резкой: как подготовить макет») — прогрев Tier-3 интентов; у нас блога нет.

## Пообъектный гэп-аудит хаба (метод)

Отдельный от общего прогона режим: берём ОДИН наш хаб и сверяем его начинку со страницами-аналогами всех конкурентов костяка.

**Процесс:**
1. Снять baseline нашего хаба: `WebFetch` нашей страницы (`printmos.ru/<slug>/`) → бумаги/материалы + опции калькулятора + функционал.
2. Разбить костяк (12 хостов) на 3 батча по 4, запустить 3 дешёвых воркера (`Agent`, model=haiku, general-purpose, WebFetch/WebSearch). Каждому передать наш baseline и просить помечать `[НОВОЕ]` то, чего у нас нет. ~2 мин, ~40k токенов/батч.
3. Свести в таблицу с колонкой «у скольких из 12», сгруппировав: **бумаги / опции / функционал**. Сортировать по частоте — частое = приоритет.

**Пилот — Визитки (2026-07-15).** Главные гэпы:
- _Функционал (важнее бумаг):_ онлайн-конструктор макета + библиотека шаблонов по отраслям — у ~8/12, у нас нет (конверсия + SEO-сетка «визитки для врача/юриста…»). Блок доставки с ценами — 7/12. Срочность <1 дня — 5/12.
- _Опции (быстрые доборы):_ каширование (~6), вырубка/фигурная резка (~6), объёмный/3D-лак (5), размер 90×55 (3), квадратные (2), шелкография/Pantone/3D-фольга (2).
- _Бумаги (вопрос закупки):_ каширование, Touch Cover (~6), Majestic (5), крафт (5), прозрачный/цветной пластик (3), Plike/калька/Sirio Pearl (2), плотности до 400.

**Аудит — Брошюры + каталоги (2026-07-16).** 12 конкурентов, 3 воркера. Главные гэпы:
- _Опции-реюз (дёшево, механизм уже в базе):_ тиснение/конгрев + УФ-лак на обложке (~5/12) — опции уже заведены на визитках, привязать junction к обложке; дизайнерская бумага обложки (**6/12**) — материалы есть; квадратные форматы 148×148/210×210 (4/12) — добавить size-пресеты.
- _Переплёт (производство):_ болты/винты (**6/12** — топ-гэп, новый тип bindings), твёрдый/hardcover (3–4/12), швейно-клеевой/КШС (3/12), евроскоба + КБС на PUR (pmg), кольца (2).
- _Резка:_ вырубка/фигурная обложка (4/12) — multipage не поддерживает `allow_complex`, нужен отдельный механизм + вопрос производству.
- _Функционал (cross-cut, не только брошюры):_ онлайн-конструктор + шаблоны (4/12) → уже в `TECHDEBT.md`; дизайн/вёрстка как явная услуга (4/12).
- _Посадочные:_ `/catalogs` критически тонкий (1 кластер) — **приоритет №1**; отраслевые брошюры (недвижимость/авто/медицина), методички/авторефераты (copy), тех.паспорта/инструкции (3/12) — Tier-2/3 интенты.
- _Уже есть (не гэп):_ soft-touch ламинация, голографич. фольга, свой размер; каталоги/меню/журналы/буклеты/фотокниги — отдельные наши хабы, не «дыра» брошюр.
