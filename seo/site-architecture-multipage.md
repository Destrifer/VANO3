## БРОШЮРЫ — `/brochures` (multipage)

Дизайн/цена/тираж/цветность — Tier‑3 вхождения. STANDALONE: буклеты/каталоги/книги.

## БУКЛЕТЫ + ЛИСТОВКИ + ФЛАЕРЫ — `/booklets` (sheet + фальц)
Ядра: буклеты 669 / 78 848; листовки‑флаеры 1175 / 95 074. Продукт `booklets`, `strategy=sheet` с `fold_types`.
**Ключ:** добавлен фальц **«Без сложения» (folds=0, idx5)** + гард в движке (биговку не считаем при folds=0) → листовка/флаер выражаются как буклет без сложения.
**Tier‑1 (6 плиток):**
- `leaflet` листовки — `{foldTypeIndex:5, sizeIndex:0(A4)}` (главный, 35k)
- `flyer` флаеры — `{foldTypeIndex:5, sizeIndex:4(Евро)}` (анти‑каннибализация по размеру)
- `fold-euro`(foldTypeIndex:1) · `fold-garmoshka`(2) · `fold-book`(0) · `urgent`
**Миграция:** продукты `leaflets`, `flyers` сняты с публикации (стали кластерами). Раздача/распространение листовок — EXCLUDE (не наша услуга). Хаб = booklets по решению владельца (буклет = суперсет).
