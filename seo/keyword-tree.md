Источник: Я.Вебмастер «Подбор запросов и анализ рынка», **регион — Москва**, 08.06.2026.
Сиды (4): `визитки`, `печать визиток`, `изготовление визиток`, `заказ визиток`.
Сырьё (master): **3152 уникальных запроса**. Числа — **спрос/клики**. Данные: `data/queries_vizitki_master.csv`.

### Решения-принципы
- **Гео «москва / в москве»** — наш регион, **не отдельные страницы** (в отличие от coral).
- **Цена «недорого / дешево / цена»** — **вхождения/блоки** на хабе, не посадочные (каннибализация).
- Ветки 13–14 — параметры конфигуратора → текстовые секции на хабе, не URL.

### Ветка «Срочные» — разложена (сид `срочная печать визиток`)

Одна посадочная **«Срочная печать визиток»** (спрос ~9.7k), внутри — блоки/H2 (НЕ отдельные страницы, спрос мелкий → каннибализация):

| Под-интент | ~Спрос/Клики | Решение |
|---|---|---|
| срочная печать / визитки срочно / срочное изготовление (ядро) | ~7.2k / ~6.1k | H1 страницы |
| **за час / за 1 час** | ~870 / ~780 | блок/H2 «за 1 час» |
| оперативная / быстро (синонимы) | ~600 / ~480 | в текст (синонимы) |
| **за 1 день** | ~170 / ~140 | блок «за 1 день» |
| **круглосуточно** | ~140 / ~120 | блок «круглосуточно» |

Гео «москва» — наш регион. Бренды (а цифра, принтмастер, гарант) — в BRANDED.

### Ветка «Тиснение / фольгирование» — разложена (сиды `визитки с тиснением`, `визитки с фольгированием`)

Одна **флагман-посадочная** (спрос ~4.3k, клики ~4.1k). «Фольгирование» = «тиснение фольгой» — **синонимы в один интент** (ровно пример из 05 §79). Внутри — блоки:

| Под-интент | ~Спрос/Клики | Решение |
|---|---|---|
| тиснение (с тиснением, тиснение визитки) | ~2.2k / ~2.0k | ядро / H1 |
| фольгирование (= тиснение фольгой) | ~1.2k / ~1.3k | синоним в H1/текст |
| золотое / серебряное тиснение (цвет фольги) | ~0.9k / ~0.7k | блок «цвета фольги» |
| конгрев | ~0.15k / ~0.14k | блок «конгрев» |
| термоподъём | ~0.1k / ~0.1k | блок (имитация конгрева) |

### Ветка «Пластиковые / прозрачные» — разложена (сиды `пластиковые визитки`, `прозрачные визитки`)

Посадочная **«Пластиковые визитки»** (спрос ~2.0k), внутри блок «прозрачные» (под-вид пластика).

| Под-интент | ~Спрос/Клики | Решение |
|---|---|---|
| пластиковые / на пластике | ~2.0k / ~1.7k | H1 |
| прозрачные (прозрачный/матовый ПВХ) | ~0.35k / ~0.3k | блок «прозрачные» (отдельная только если хватит контента) |

> Мусор в этой ветке: «альбом/подставка/визитница для хранения визиток» — это товары хранения, не печать → в стоп-лист (добавить `визитница`, `альбом для`, `подставка для`).

### Уточнение по master (4 сида) — что подтвердилось и что нового

**Новое / важное:**
- **«с доставкой»** — сильный модификатор: спрос **~3 580** (заказ/визитки с доставкой, онлайн-заказ с доставкой). Это не отдельная посадочная (фулфилмент, не продукт) → **USP-блок на хабе** + в title/описании.
- **«заказать визитки онлайн»** (585/306) — легит онлайн-заказ печати (НЕ генератор) → блок «онлайн-заказ» на хабе.
- **Офсетная печать** — крупный самостоятельный интент: `типография офсет москва` 4424/2859, `офсетная печать в москве` 314/320. Уровень почти хаба (шире визиток) → ветка «технология: офсет», возможно отдельный хаб офсетной печати позже.

## Исключаем (NONTARGET) — чужой бизнес

сайт-визитка / создание-разработка-продвижение сайтов (~80k+ спроса!), онлайн-генераторы (создать визитку онлайн, конструкторы), ИИ/нейросети для визиток, электронная/цифровая/видео-визитка, приколы (игра в кальмара, американский психопат, визитка яроша, матрица судьбы), бух-вопросы. Полный список — `category=NONTARGET` в CSV.

## Гайды (INFO) — инфо-кластер, отдельно от коммерции

шаблон/образец визитки, как сделать визитку, размер/формат визитки (спрос 842!), примеры визиток, как распечатать на А4 / раскладка, подготовка макета к печати. → блог/гайды, перелинковка в хаб.

## Drill-down: статус и вывод о насыщении

✅ Обработано **9 сидов**: `визитки`, `печать/изготовление/заказ визиток`, `срочная печать`, `тиснение`, `фольгирование`, `пластиковые`, `прозрачные`. Master = 3320 запросов.

**База близка к насыщению** (анализ пересечений):
- 77% запросов встречаются лишь в одном сиде, но узкие модификаторы почти не дают нового: «визитки с тиснением» — **13% эксклюзива** (87% уже были в широких сидах), 4 модификатор-сида суммарно +151 запрос.
- **Широкие сиды** (печать/изготовление/заказ) дают 51–65% нового — они и есть основной источник хвоста.
- → **Дробить каждый микро-модификатор отдельным сидом не нужно** — они уже внутри широких выгрузок. Diminishing returns.

**Осталось добрать (ещё дают новое, т.к. смежные интенты):**
`дизайн визиток` (услуга дизайна), `офсетная печать` (крупный самостоятельный интент). После них ядро визиток можно считать собранным.

Обработка master-набора (любое число файлов):
```
python _pipeline.py vizitki_master <file1.xlsx> <file2.xlsx> ...
```
