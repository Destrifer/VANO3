# Архитектура раздела ВИЗИТКИ (ТЗ на страницы)

Из дерева интентов (`keyword-tree.md`, master 9 сидов, Москва) → решение по страницам.
Уровни: **ХАБ** → **Tier-1 (плитки на хабе)** → **Tier-2 (в sitemap, без плитки, хвост)** → **Tier-3 (только вхождение)**.

## Фильтр возможностей (что мы реально можем)

| Можем | Ограниченно | НЕ можем |
|---|---|---|
| Цифровая печать, тиснение/фольга, пластик, магнитные, металл, ламинация, лак, дизайнерские материалы, срочно, вырубка | **Дизайн** (делаем, но не основное, нет штат. дизайнера/конструктора дизайна) → не флагман | **Офсет** — нет оборудования |

→ **Офсетные запросы — нецелевые** (вычёркиваем, как сайт-визитки), несмотря на спрос (`типография офсет москва` 4424 — не наш).
→ **Дизайн** — скромный блок «поможем с макетом» + гайд, не посадочная-флагман.

## 1. ХАБ — `/vizitki` «Печать визиток»

H1: **Печать визиток в Москве**. Цель: ранжироваться по голове + развести на плитки + довести холодного до цены.

### Вхождения в текст хаба (НЕ отдельные страницы — синонимы/параметры головы)

| Группа вхождений | Запросы | Куда на странице |
|---|---|---|
| Синонимы головы | изготовление визиток, визитки на заказ, заказать визитки, производство визиток, типография визитки, напечатать/распечатать визитки | H1/intro/H2 естественным текстом |
| Цена | визитки цена, недорого, дёшево, стоимость, сколько стоит | блок «Цены» + таблица тиражей |
| Тираж | 100 шт, 1000 шт, опт | таблица цен по тиражам |
| Гео | москва, в москве, мск | в title/H1/тексте (наш регион, НЕ страницы) |
| Доставка (спрос ~3.6k!) | с доставкой, онлайн-заказ с доставкой | **USP-блок «Доставка по Москве/РФ»** |
| Онлайн-заказ | заказать визитки онлайн | блок «Рассчитать и заказать онлайн» (конфигуратор) |
| Стороны | двусторонние, односторонние | параметр конфигуратора + строка характеристик |

### Плитки на хабе (ссылки на Tier-1)
Срочные · С фольгированием/тиснением · Пластиковые · На дизайнерской бумаге · Магнитные · Металлические · Круглые/фигурные · VIP-премиум.

## 2. Tier-1 — посадочные с плиткой на хабе

Для каждой: **своя механика/UX** (не «то же с заголовком»), опираясь на наши активы — конфигуратор с мгновенной ценой, `Preview.vue` (фольга/ламинация/материал), preflight, расчётная дата готовности, скачиваемый шаблон, свотчи бумаг (`SwatchPalette`).

| Стр. (URL) | H1 | ~Спрос/Клики | Чем уникальны (механика/UX-гипотеза) | Вхождения внутри |
|---|---|---|---|---|
| `/vizitki/srochnye` | Срочная печать визиток | ~9.7k / ~8.4k | **«Успеем сегодня»**: cutoff-таймер до приёма + расчёт **точного времени готовности** (не «3-5 дней»), слот на самовывоз сегодня | за час, за 1 день, круглосуточно, оперативная, быстро |
| `/vizitki/foilirovanie` | Визитки с фольгированием (тиснением) | ~4.3k / ~4.1k | **Интерактивный preview фольги** (золото/серебро/цвет) на макете в конфигураторе; факт «на какую бумагу фольга не ложится» | тиснение, золотое/серебряное, конгрев, термоподъём |
| `/vizitki/plastikovye` | Пластиковые визитки | ~2.35k / ~2.0k | Preview прозрачного/матового ПВХ + шаблон под пластик + честно о толщине/сроках | прозрачные, матовый/серебро/золотой пластик |
| `/vizitki/dizajnerskie` | Визитки на дизайнерской бумаге | ~1.7k / ~1.6k | **Живые свотчи бумаг** (фото фактуры, `SwatchPalette`) + подсказка совместимости с фольгой/лаком | картон, кашированные, лён, тач-кавер/touche cover, софт-тач, плотная бумага |
| `/vizitki/magnitnye` | Магнитные визитки (на магните) | ~1.1k / ~0.9k | Preview + спецификация магнитного слоя, «визитка-магнит на холодильник» как сценарий | на магнитах, магнит на холодильник |
| `/vizitki/metallicheskie` | Металлические визитки | ~1.1k / ~0.9k | Галерея реальных металл-карт + варианты (травление/гравировка), честные сроки/MOQ | из металла, металлические карточки |
| `/vizitki/figurnye` | Круглые и фигурные визитки (вырубка) | ~0.44k / ~0.3k | **Выбор формы/контура в preview** (круг/скругление/вырубка) | круглые, со скруглением, сложной формы, с вырубкой |
| `/vizitki/vip` | VIP / премиум визитки | ~0.5k / ~0.4k | Кураторская подборка «премиум» (фольга+дизайн.бумага+кромка) — конструктор предзаполнен премиум-опциями | элитные, эксклюзивные, премиум |

> ⚠️ Дифференциация — **гипотезы**. Финал подтвердить: (а) факты от тебя (что реально умеем уникально), (б) разбор страниц конкурентов (projectorprint/тиснение, digital-printing/срочно, kopirka, izoproekt).

Закрывают микро-НЧ, доступны через sitemap/внутренние ссылки, но не на главных плитках. Делать **только если есть уникальный контент** (своя специфика/галерея/FAQ), иначе → Tier-3.

## 4. Tier-3 — ТОЛЬКО вхождение в текст (каннибализация / параметр)

Не страницы. Уже расписаны как вхождения хаба или внутренних страниц:
- голова-синонимы, цена/недорого, тираж, гео, доставка, двусторонние → **хаб**;
- за час/за 1 день/круглосуточно/оперативно → **`/srochnye`**;
- золото/серебро тиснение, конгрев, термоподъём → **`/foilirovanie`**;
- прозрачные, матовый пластик → **`/plastikovye`**;
- глянцевые (спрос ~1, Москва) → вхождение, страницы НЕ надо.

## 5. Исключить (нецелевое для нас)

- **Офсет** (нет оборудования): офсетная печать визиток, типография офсет — не наши.
- **NONTARGET** (из дерева): сайт-визитка, генераторы, ИИ, электронная/видео-визитка.
- **Товары хранения**: визитница, альбом/подставка для визиток.
- **Дизайн как флагман**: только блок «поможем с макетом» + гайд (нет ресурса под полноценную услугу).

## Сверка с Coral-Print (повторная проверка)

Перепроверены хаб + под-деревья (цифровая, VIP, дизайн). Вывод: **типы страниц мы не упустили, на материалах мы полнее.**

**Их хаб-плитки (6):** Простые · Круглые · Сложной формы · Фольгирование · На пластике · VIP.

| Аспект | Coral | Мы | Вывод |
|---|---|---|---|
| Магнитные / металлические | **нет** (хотя спрос ~1.1k каждый) | есть как плитки | **наша фора** (их пробел) |
| Конгрев / термоподъём | под-страницы VIP | вхождения в `/foilirovanie` | сгруппировано по технике (логичнее) |
| Аудиторные ниши (массажист…) | под-хаб `/dizain-vizitok/*` + свой конструктор дизайна | отложено (нет дизайнера) | осознанный пропуск, не упущение |
| «Простые визитки» | отдельная плитка | = дефолт хаба | отдельная страница не нужна |
| Круглые + Сложной формы | 2 плитки | 1 плитка «фигурные/круглые» | у них пере-дробление (наш спрос низкий) |

**Их фишки = UX/механика (не страницы) — взять на вооружение:**
- **Галерея с табами** (Подборка / Стандартные / Премиальные / С фольгированием) — фильтруемое портфолио. У нас `getWorks` + фильтр уже есть → добавить табы на хабе/посадочных.
- **Виджет отзывов Google/Яндекс** (у них 408 отзывов, 4.9) + механика «оставь отзыв → скидка 10%». Нам — собирать и выводить отзывы (соц.proof + AggregateRating schema, которой у них нет).
- **Плитки с ценовой лесенкой** «от X ₽» (490→690→1680→3000→3800) — подтверждает наш план: «от X ₽» на каждой плитке.
- **Конструктор дизайна** `/constructor` (НЕ путать с нашим конфигуратором цены) — их преимущество для аудиторного хвоста; у нас нет, пока не делаем.

**Их слабости (наша фора, подтверждено):** ноль JSON-LD и FAQ на всех страницах; не закрыты магнитные/металлические. NB: фора по FAQ — про вопросный хвост + AEO, а **не** про rich-сниппет (Google убрал FAQ rich results с мая 2026). FAQ-блок у нас опционален per-page (заполнен → показывается), наполнение — из выдачи/Wordstat/подсказок (05 «Откуда брать FAQ»).

## Что нужно от тебя / следующий шаг

1. **Подтвердить уникальные механики** по Tier-1 (колонка «чем уникальны») — что из этого реально можем, что добавить. Это «факты владельца» для будущих текстов.
2. **Разбор конкурентов по топ-плиткам** (срочные, фольга) — могу пройтись по их страницам через браузер и вытащить, что у них есть/нет, чтобы наша механика была реально лучше, а не такой же.
3. После подтверждения — финализировать список URL и перейти к реализации хаба (Этап 3 из `06-seo-implementation.md`) на этих данных.
