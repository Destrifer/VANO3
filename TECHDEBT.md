    ПРОВЕРИТЬ НА ЛОКАЛИ, что наша публичная read-политика (сайт без токена) = Core,
    иначе сборка сайта ляжет. Grace period 30 дней после апгрейда.
  - **`IP_TRUST_PROXY` теперь default=false** — мы за Caddy, в прод-env добавить
    `IP_TRUST_PROXY=true` (иначе rate-limit/логи видят IP Caddy).
  - `/server/health` закрыт для анонимов — наш compose дёргает только pg_isready, ок.
  - UI-редизайн админки; breaking для расширений/тем — у нас их нет.
  - **Репетиция на локали ✅ 2026-07-02:** миграции 11.17.4→12.1.1 прошли за ~3 с
    без ошибок; публичный API без токена РАБОТАЕТ на 12 (наши плоские права =
    Core-совместимы, кастомных правил у нас нет — проверено по /permissions);
    запись/удаление ок; админка ок; откат (restore дампа + пин 11.17.4) отработан.
  - **⛔ Блокер прода: у нас 30 коллекций > лимита Core (25).** Без ключа через
    30 дней grace Directus начнёт блокировать эндпоинты (данные не удаляет).
    → **Владелец: получить бесплатный OIG-ключ** (directus.com/oig или
    licensing@directus.io; критерии <$5M/<50 чел — проходим) ДО апгрейда прода.
  - Порядок прод-апгрейда (после ключа): `pmos-backup` → в `.env.production`
    ключ + `IP_TRUST_PROXY=true` → бамп образа 12.1.1 → миграции → проверить
